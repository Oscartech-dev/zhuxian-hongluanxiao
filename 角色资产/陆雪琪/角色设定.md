# 陆雪琪 · 角色资产

> 青云门小竹峰首座，天琊剑持有者。第1集全篇核心人物。
> 视觉风格锚点：白衣如雪、清冷克制、剑修气韵、东方古典出尘。

## 一、角色档案

| 项 | 设定 |
|----|------|
| 来源 | 萧鼎《诛仙》原著 / 《红鸾笑》番外 |

| 职位 | 首座（修真大战后接任） |
| 年龄感 | 二十岁上下，长期修行者 |

| 性格 | 克己、温柔、坚定；从不出声请求 |

### 第1集戏份定位

陆雪琪是第1集**唯一出场人物**，承担"先静后动"的叙事：

- 起念（独坐窗前） → 决意（抬眸） → 御剑出发 → 抵达废墟 → 行走追忆 → 发现木屋 → 认衣角 · 不说 → 重逢微笑

全片无台词，只靠眉目、步态、衣袂、剑光和风铃叙事。第1集出场的陆雪琪是**刚刚接任首座、还没被红尘消磨**的清冷少女剑修——不是情绪外露的少女，而是**一切都在克制里的温柔**。

## 五、风格锚点（贯穿所有镜头）

- **光线**：暖橙清晨逆光（sunrise warm backlight），金色日出，金色边光打在发梢和衣领
- **配色**：白 + 浅青 + 碧蓝剑光，画面整体淡雅；不要大红大紫
- **氛围**：清冷、克制、有重量；东方仙侠 3D 动画电影质感（接近腾讯诛仙动漫）
- **动作**：小幅、少而精；衣袂轻动、剑光一闪、眉目一动；不做大动作
