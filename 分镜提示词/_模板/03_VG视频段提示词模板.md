# 03 · VG 视频段提示词模板

> 基于第1集 VG-01「小竹峰起念与出门」跑通 Seedance 2.0 试验沉淀。
> 写新 VG 时直接复制本骨架，再按镜头内容填实即可。

---

## 一、VG 提示词整体结构（三段式）

```
【基本设定】          ← 身份层（人物 + 场景 + 声音，一句话锁身份）
  人物：...
  场景：...
  声音：...

【氛围与面质】        ← 风格层（全片统一，不随镜头变）
  风格核心：...
  视觉基调：...
  色彩与影调：...

分镜一：00:00-00:04   ← 执行层（每个分镜一组）
  景别：...
  构图：...
  机位与镜头：...
  运镜手法：...
  画面内容：...
  声音：...

分镜二：00:04-00:07
  ... 同样 6 字段

分镜三：00:07-00:11
  ... 同样 6 字段

分镜四：00:11-00:14
  ... 同样 6 字段
```

## 二、各字段规范

### 2.1 【基本设定】

| 字段 | 写什么 | 不写什么 |
|------|--------|---------|
| 人物 | 脸型 / 眉眼 / 发型 / 体型 / 服装 / 气质（一句话定位，不长篇） | 心理活动、动机、对白 |
| 场景 | 关键建筑 / 道具 / 光线基调（不写人物动作） | 人物姿态、人物动作 |
| 声音 | 是否用配乐、是否同期声 | 具体音效用词（用分镜的声音字段） |

### 2.2 【氛围与面质】

| 字段 | 写什么 | 不写什么 |
|------|--------|---------|
| 风格核心 | 动画厂名 / 风格 / 美学关键词；本 VG 的整体情绪节奏 | 镜头级描述 |
| 视觉基调 | 焦段分布（广角 / 中景 / 长焦）、运镜节奏、光影总规 | 具体某个镜头的描述 |
| 色彩与影调 | 主色 + 点缀光 + 时间感 + 质感（手绘 / 写实 / 旧感） | 镜头级色彩 |

### 2.3 分镜 6 字段（每镜必填）

| 字段 | 必含内容 | 易错 |
|------|---------|------|
| 景别 | 大全景 / 中景 / 近景 / 特写 / 微距 等 | ❌ 不要写"全景近景"含糊词 |
| 构图 | 主体 + 环境元素的位置关系 | ❌ 不要只写"主体居中" |
| 机位与镜头 | 机位高度（米）+ 焦段 + 景深 + 观察角度 | ❌ 不要漏机位高度 |
| 运镜手法 | 运动方式（推 / 拉 / 横移 / 摇 / 跟）+ 速度 + 起止 | ❌ 不要只写"缓慢推进"无速度感 |
| 画面内容 | 这一镜发生的视觉事件，必须与前 3 字段一致 | ❌ 背身镜头不要写面部特写 |
| 声音 | 同期声 / 配乐 / 静默；不写"声音设计" | ❌ 不要写"动听"等感性词 |

## 三、跨段一致性约束

写新 VG 时，【基本设定】和【氛围与面质】要与前序 VG 同步：

- 同集内【氛围与面质】可只在第1个 VG 写完整，后续 VG 只写"延续 VG-0N 氛围"避免重复
- 【基本设定】每次都要写，但可只写本段新增人物/场景
- 分镜时间码 6 字段每一镜都必填，不要漏

## 五、VG-01 真实范例（可直接复用结构）

完整文本如下，已跑通 Seedance 2.0 试验，复制时改人物名 / 场景名 / 分镜内容：

```text
【基本设定】
人物：陆雪琪 （小竹峰首座，白衣女剑修），鹅蛋脸、柳叶弯眉、凤眼微挑、冰蓝色瞳孔、肤白如雪；黑发及腰、玉簪盘起、长鬓发垂落；身形修长挺拔（7.5-8 头身），纯白长裙、纱质外袍、银丝暗纹、白丝绦；清冷克制气质。脸型、五官、发饰、服装以陆雪琪四视图角色母版为准。
场景：小竹峰外景 ——青蓝竹林、溪流、小巧弧形拱桥、青瓦飞檐、回廊残灯、晨雾笼罩，林深处藏仙门雅舍；居所室内 ——圆窗月洞门透光、竹帘半垂、素色木案、卷轴、青瓷茶盏。两处共用清晨光线（青蓝晨光 + 暖黄残灯点缀），整体清幽出尘，绝非农舍或宫阙。
声音：仅保留同期声和对白，无配乐。

【氛围与面质】
风格核心：诛仙仙侠剑修、腾讯诛仙动画剧集3D国漫、玄机科技建模路线、清冷克制美学。从孤清独坐到决意起身，情绪推进以眉目微动、指节收紧、衣袂轻扬等细节动作为主，而非夸张表情；分镜三温柔的内心独白（同期录音，呼吸般的低语）。镜头美学拒绝炫技——慢推、慢停、慢呼吸，让观众的注意力一直停在陆雪琪的心事上，而不是镜头本身。
视觉基调：16:9宽幅。24mm广角建立小竹峰大环境，不急于交代人物；室内段35mm～50mm中景跟拍陆雪琪独坐与起身；特写及近景段85mm定焦，光影透过圆窗和纱帘落在她侧脸和衣领上。所有运镜必须克制，机位移动量极小，靠被摄物自身的微动（纱帘、残灯、竹影）传递时间的缓慢流动。
色彩与影调：以青蓝冷晨光为画面主色层——竹叶的青、晨雾的灰蓝、远山的墨青、室内石壁的冷灰，构成清冷基底。回廊残灯的暖黄光作为唯一暖色点缀，冷暖对比克制，不绚丽不喧哗。画面质感接近3D动画中手绘质感的柔和笔触，雾气有体积感但不过度梦幻，衣料和竹木有经年使用的旧感，杜绝过曝、杜绝塑料皮肤、杜绝大红大紫。

分镜一：00:00-00:04
景别：电影感大环境全景。
构图：小竹峰竹林、溪桥、雅舍、圆窗、回廊残灯占据画面主要空间，人物不出现。
机位与镜头：平视略高机位，约 1.6-2.0 米，模拟人远处回廊望向山门；24mm 广角，深景深。
运镜手法：极慢横移，移动量小；镜头自左向右沿外景横向滑过，像清晨刚醒来的目光慢慢扫过山门。
画面内容：天色刚亮，青蓝晨雾压在竹林和溪桥之间，几盏回廊残灯还微微亮着。溪水很轻，竹叶几乎不动。几只早起的鸟儿从林梢扑翅掠起，斜斜飞过画面，远处传来几声清脆鸟鸣——时辰尚早，整座小竹峰还未苏醒。镜头不急于交代情节，只先交代孤清、幽静、仙门清晨的气息，为陆雪琪的心事铺底。
声音：晨风、溪水、竹叶声、鸟儿扑翅声、清脆鸟鸣。

分镜二：00:04-00:07
景别：室内中景，陆雪琪背影。
构图：圆窗、纱帘、木案、卷轴、青瓷茶盏在画面中形成安静的室内层次，陆雪琪坐在窗前偏中位置，白衣背影清瘦挺直。
机位与镜头：平视略低机位，约 0.8-1.0 米；35mm-50mm 中景；中景深，前景纱帘虚化、中景人物背影、背景圆窗明亮。
运镜手法：镜头非常慢地向内推，不打扰人物，像在靠近一段无人知晓的心事。镜头始终保持在陆雪琪身后或侧后方，不绕到正面，不暴露面部。
画面内容：陆雪琪独坐窗前，白衣落在晨光里，背影清瘦挺直。她望着窗外很久没有动——颈椎微沉、肩线轻压，像压着一段放不下的心事。她的手垂在木案旁，指尖偶尔轻触卷轴边缘或青瓷茶盏，又松开。纱帘被晨风轻轻推动，掠过她肩侧。背影克制中透出牵挂，但不展示面部表情，避免 AI 误读为夸张悲伤、哭泣或面部特写。
声音：屋内安静，纱帘轻响，呼吸声很轻。

分镜三：00:07-00:11
景别：手部特写、面部近景连续组接。
构图：先近景特写拍她垂在木案旁的手，指尖轻触卷轴边缘或青瓷茶盏；再切到她的眼睫、唇线和侧脸进行特写，面部表情有呼吸感
机位与镜头：手部特写段约 0.6 米俯拍机位，浅景深虚化背景；面部近景段约 1.2 米平视机位，85mm 定焦，浅景深虚化圆窗与纱帘。
运镜手法：慢镜头，85mm 近景，轻微推近；每一个动作都要慢，停顿要长，情绪靠细节出现。
画面内容：她的手指微微收紧，又松开，像在压住一个念头。晨光掠过她的眼睫，她垂眼片刻，唇线轻轻绷住。她低声喃喃，仿佛在对一个不在场的人说话：小凡……是你吗？你在哪里？说完后，她的呼吸微微一滞，眼底的犹豫慢慢沉下去，眼神反而变得清澈而坚定。人物面部严格参考陆雪琪四视图角色母版，白色仙门服饰、黑发、玉簪、清冷气质保持一致。
声音：低语，不加字幕。

分镜四：00:11-00:14
景别：室内中景到门口方向的跟拍。
构图：陆雪琪从窗前起身，白衣袖摆掠过木案和纱帘，圆窗与清晨竹影留在身后。
运镜手法：前半秒极慢停顿（她在窗前骤然抬眼、神情有变），起身后轻微跟拍，速度略微加快；镜头停在她走向门外的侧影或背影处，为下一镜天琊出鞘、御剑飞行留接口。
画面内容：她在窗前骤然停住——瞳仁微张、唇线一抿、眉心一紧，像一个散落的念头突然被自己捕捉到。下一瞬，她干净利落地起身，动作不再迟疑。不是冲动，不是惊慌，而是一个被压了很久的答案突然浮出水面：去草庙村，看他一眼。她向屋外走去，衣袂带起一阵轻风，屋内的晨光被她留在身后。
声音：衣料轻响、脚步声、竹叶风声稍起。
```

## 六、易错点速查

| 错误 | 修正 |
|------|------|
| 背身镜头写面部特写 | 改为"背影中可见的肩线 / 颈椎 / 手部动作" |
| 机位只写"平视"无高度 | 加米数（如"平视略低，约 0.8-1.0 米"） |
| 焦段只写"中景" | 加焦段（"35mm-50mm 中景"） |
| 时间码写"00:00-04s" | 用 `HH:MM-SS` 格式（"00:00-00:04"） |
| 运镜写"缓慢"无参照 | 加移动方向 + 速度描述（"极慢横移，移动量小"） |
| 声音写"动听 / 感人" | 改为具象音（"晨风、溪水、竹叶声、鸟鸣"） |
| 把文件路径写进提示词 | 路径归 HTML 参考资产区，提示词只保留视觉身份描述 |
| 风格锚点写"现代、霓虹" | 反向词已写"杜绝过曝、塑料皮肤、大红大紫"，正面词围绕"青蓝晨光、手绘质感、旧感"展开 |
