# 03A · Seedance 视频段增强写法

> 基于 VG-01 已跑通模板新增，不替代 [03_VG视频段提示词模板.md](./03_VG视频段提示词模板.md)。
> 使用原则：VG-01 基线模板保持稳定；从 VG-02 开始，可按本文件选择性增强。

---

## 一、增强版总原则

Seedance 视频提示词不能只是“让参考图动起来”。每个 VG 应被写成一个完整电影片段，包含：

- 清晰时机：0-5s / 5-10s / 10-15s 的节奏推进
- 摄影机运动：机位高度、焦段、运动路径、速度变化
- 主体动作：人物、道具、环境如何进入或变化
- 氛围与光线：天气、色温、逆光、雾、阴影和材质
- 情绪落点：最后 1-2 秒停在一个明确的电影瞬间

固定增强句：

```text
参考资产只作为人物、场景、美术和空间一致性依据，不要简单把参考图做成动画；请根据剧情补足缺失的动作、转场、环境变化和情绪节奏，形成一个完整电影片段。
```

## 二、三段节奏法

适用于 10-15 秒 VG；短于 10 秒时可压缩为两段。

| 时间段 | 功能 | 写法 |
|--------|------|------|
| 0-5s | 建立环境、主体状态、视觉氛围 | 先让观众知道地点、方向、人物状态 |
| 5-10s | 增加动作、信息、情绪或压力 | 让主体行为改变，或让画面信息推进 |
| 10-15s | 最强电影时刻与情绪落点 | 给出最清晰的画面记忆点，并为下一 VG 留接口 |

短视频段模板：

```text
节奏结构：本 VG 约 [X] 秒，采用 [两段式 / 三段式]。前段 [建立什么]，中段 [推进什么]，末段 [落在什么情绪画面]。
```

## 三、每镜新增“镜头目的”

在每个分镜中可加入：

```text
镜头目的：[让观众感受到什么 / 交代什么信息 / 推动什么情绪]
```

常用写法：

- 交代空间关系：让观众确认人物从哪里来、要去哪里
- 情绪转折：让观众看见人物从犹豫到决定
- 叙事线索：让观众看见木屋、炊烟、衣角等关键信息
- 压力释放：让观众从紧张转入安静、从飞行转入俯瞰
- 留接口：让最后画面自然接到下一 VG

## 四、运镜增强词库

### 4.1 适合《红鸾笑》当前气质的常用运镜

| 类型 | 关键词 | 适合场景 |
|------|--------|----------|
| 缓慢推进 | Slow Push-in / Dolly In / 固定轨道慢推 | 人物心事、静态空间、重要线索靠近 |
| 缓慢拉远 | Slow Pull-back / Dolly Out | 从个人情绪拉回命运或大环境 |
| 主观视角 | POV / First-person Perspective | 御剑飞行、人物视线、发现线索 |
| 航拍俯瞰 | Aerial / Drone shot / High Angle | 青云山、草庙村全貌、云海建立镜头 |
| 低角度仰拍 | Low Angle shot / Tilt-up | 天琊出鞘、人物从静到动、仙剑起势 |
| 跟拍 | Tracking shot / 侧面跟拍 / 背后跟拍 | 行走、穿过废墟、靠近木屋 |
| 匹配剪辑 | Match cut / 视觉匹配 / 无缝转场 | 圆窗到太阳、衣角到衣袂、剑光到云光 |
| 升格慢动作 | 120fps slow motion / 时间凝固感 | 衣角扬起、剑鸣一瞬、眼神停顿 |

写法公式：

```text
运镜手法：[运镜类型]，从[起点画面]到[终点画面]，[速度变化]，[情绪目标]。避免无意义晃动，运动必须服务叙事。
```

## 五、光影色彩增强词库

适合第1集清晨仙侠质感：

- 清晨青蓝雾气
- 暖金日出逆光
- 云层边缘光
- 发丝轮廓光
- 竹叶投影
- 体积雾 / 山间薄雾
- 冷暖对比克制
- 低饱和国风色彩
- 3D 国漫柔和手绘质感

谨慎使用：

- Golden Hour Backlight：适合飞行、重逢、暖色记忆
- Film Grain / 胶片颗粒：可少量用于回忆感，但不要破坏国漫干净质感
- Low Key Lighting：适合悬疑、夜戏、压迫感，不适合小竹峰清晨段
- Teal and Orange：适合现代战争/动作，不作为本剧默认色彩

## 六、未来可用但暂不加入当前 VG-01 的技巧

这些技巧不适合 VG-01 的清冷起念段，但后续若出现动作、悬疑、惊险、压迫或强风格段落，可再启用。

| 技巧 | 适合未来场景 | 当前不加入原因 |
|------|--------------|----------------|
| Dutch Angle / 荷兰角 | 精神失衡、幻境、恐惧、巨大变故 | VG-01 情绪克制，不需要倾斜构图 |
| Crash Zoom / 快速变焦推拉 | 突然发现、惊吓、强反转 | VG-01 不是惊吓段，会破坏慢节奏 |
| High-speed Chase / 高速追逐动态模糊 | 追逐、逃亡、战斗、奔逃 | 当前没有追逐动作 |
| Dolly Zoom / 希区柯克变焦 | 心理恐惧、空间压迫、巨大震惊 | 第1集前段更偏怀旧与牵挂 |
| Low Key Lighting / 暗调低调摄影 | 夜探、危机、秘密、恐怖感 | 当前是清晨仙门，不应黑暗化 |
| Teal and Orange / 好莱坞青橙色调 | 现代动作、战争、强商业感 | 容易偏离诛仙国风仙侠 |
| 强胶片颗粒 / 90年代录像质感 | 回忆、旧梦、凡间旧事插叙 | 可用于回忆，不作为主线默认质感 |
| Time-lapse / 延时摄影 | 大时间流逝、云海变幻、城市/天象 | 本集目前以实时叙事为主 |
| Worm's-eye View / 虫视镜头 | 压迫性巨物、神像、强敌登场 | 当前人物关系不需要压迫感 |

## 七、VG-02 试验方向

VG-02「天琊出鞘与云海俯瞰」适合测试以下增强技巧：

- 低角度仰拍 / Tilt-up：用于天琊剑光亮起、陆雪琪起势
- POV 主观视角：用于御剑穿云，让观众跟随她的视线飞向草庙村
- 航拍俯瞰 / Aerial High Angle：用于青云山、云海、草庙村废墟空间关系
- 慢到快再放缓：用于起剑、穿云、抵达俯瞰的节奏变化
- 最终电影时刻：白衣剑光穿出云层，草庙村废墟在山脚雾中显现

VG-02 不建议使用：

- 大量夸张剑气
- 战斗姿势
- 近景英雄特写
- 强烈追逐镜头
- Crash Zoom
