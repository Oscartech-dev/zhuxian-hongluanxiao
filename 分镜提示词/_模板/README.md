## 复用原则

1. **先查表**：写新东西前先看是否有现成模板可套用。
2. **不重复造**：模板里的字段、命名、风格锚点已被验证有效，不要私自增删。
3. **改后同步**：如果模板因新需求需调整，先在本文件夹更新，再应用到 HTML。
4. **跨集要保持一致**：第1集确定的格式，原则上第2/3集继续沿用；如必须改，记入 `项目架构-更正版.md` 的「风格定式」。
