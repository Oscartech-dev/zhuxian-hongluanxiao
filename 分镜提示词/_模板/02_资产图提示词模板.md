# 02 · 资产图提示词模板

> 第1集沉淀下来的三大类资产生成提示词结构化模板。
> 每类资产生成前，先按对应模板写好提示词，再调用生图工具。

---

## 总原则（适用所有资产）

1. **一张资产图对应一个提示词**，避免多图混写
2. **不写人物情绪、戏剧动作**（场景母图尤其如此）
3. **写明"用途"**——后续哪个镜头会用这张图，让 AI 知道该渲染什么
4. **必须包含负面限制**，避免模型跑偏
5. **中文概念描述 + 英文生图提示词 + 英文负面限制**三段式
6. **写完入 `资产提示词库` 索引**，HTML 引用

---

## 一、场景母图（场景资产）

> 详细方法论见 `红鸾笑/场景资产/场景母图提示词技巧.md`（更完整的规则）
> 这里给的是可复制的骨架模板。

### 用途

建立空间，作为多镜头的母版裁切源；不含人物与戏剧动作。

### 推荐提示词结构

```text
项目与用途：第N集 第N幕 / 场景母图 / 用于[镜A, 镜B, 镜C]裁切
参考方向：（从已有参考图提取的有效元素，可选）
画面规格：16:9 横版，[时间段如清晨/午后]，东方仙侠 3D 动画电影质感
机位：平视略高，[焦段如24mm广角]，[角度如三分之二侧前方]，[景深如深景深]
空间层次：前景 [xxx]，中景 [xxx]，远景 [xxx]
关键视觉符号：[必出现的建筑/道具/环境元素]
光线与色调：主色 [xxx]，点缀光 [xxx]，时间感 [xxx]
复用要求：留白区域 / 可裁切位置 / 后续加人物位置
负面限制：不要人物 / 不要 [错误元素] / 不要 [错误氛围] / 不要 [错误时代]
```

### 已确认模板案例

**外景** —— `小竹峰_竹林溪桥仙门雅舍清晨残灯.png`

```text
项目与用途：第1集 第一幕 / 外景母图 / 用于 VG-01 镜 1A、VG-03 起幅、VG-09 收尾
画面规格：16:9，黎明前后青蓝清晨
机位：平视略高，24mm 广角，深景深，三分之二侧前方
空间层次：
 - 前景：竹林、溪流、薄雾
 - 中景：弧形拱桥、回廊残灯
 - 远景：山门雅舍、远山晨雾
关键视觉符号：幽蓝竹林 / 溪流 / 小巧弧形拱桥 / 圆形月洞门或圆窗 / 青瓦飞檐 / 回廊残灯
光线与色调：
 - 主色：青蓝、墨绿、淡白晨雾
 - 点缀光：回廊残灯暖黄
 - 时间感：清晨（不是夜晚，也不是白天）
复用要求：桥两端、雅舍门口、圆窗区域留出可裁切空间
负面限制：不要人物、不要动物、不要农舍感、不要茅草屋顶、不要柴火堆、不要节庆红灯笼、不要现代元素
```

**室内** —— `小竹峰_居所室内圆窗清晨空镜.jpg`

```text
项目与用途：第1集 第一幕 / 室内空镜 / 用于 VG-01 镜 1B、VG-01 分镜二/三分镜背景
画面规格：16:9，黎明后青蓝晨光
机位：平视，35mm 中景，深景深，居中略偏左
空间层次：
 - 前景：圆窗月洞门、竹帘虚化
 - 中景：木案、卷轴、青瓷茶盏、人物可坐位置
 - 远景：窗外竹林晨雾透进
关键视觉符号：圆窗月洞门 / 竹帘 / 素色木案 / 卷轴 / 青瓷茶盏
光线与色调：青蓝晨光从圆窗透入 + 屋内少量暖灯
复用要求：木案前留出人物坐姿位置；圆窗外竹林细节可被镜头借用
负面限制：不要人物、不要现代家具、不要华丽装饰、不要塑料感、不要曝光过度
```

## 二、角色参考图（角色资产）

> 详细方法论见各角色目录下的 `角色设定.md`（如 `红鸾笑/角色资产/陆雪琪/角色设定.md`）
> 这里给的是通用骨架。

### 用途

定调角色脸型 / 体型 / 发型 / 服装，作为后续所有镜头合成 / 改图 / 视频的"母版"。

### 推荐提示词结构

```text
项目与用途：第N集 / 角色参考表 / 角色：[角色名]
画面规格：16:9 横版（或 1:1 / 2:3 视构图需要），同张画布
风格：[动画厂名] 建模路线 / 清冷写实系国漫美型角色

统一人物设定：[角色名]，[年龄]，性别，[派别/职位]，[武器/标志]。
脸：[脸型]，[眉形]，[眼形]，[瞳孔色]，[唇色]，[肤色]。
发型：[发色]，发长，[发型描述]，[发饰]。
体型：[身高感]，[头身比]，[体态]。
服装：[主色]，[服装类型]，[材质]，[纹饰]，[腰饰]。

视图要求：
1. 正面视图 Front View：正面 0 度，[姿态]。
2. 侧面视图 Side View：左侧 90 度，[姿态]。
3. 背面视图 Back View：背面 180 度，[姿态]。
4. 人像特写 Portrait Close-up：肩部以上，[脸部细节]，[眼神要求]。

要求：[前三个全身视图头顶脚底水平一致 / 四个子图人物一致 / 8K / 柔和光影 / 真实皮肤质感 / 等]
负面限制：[不要换脸 / 不要尖下巴 / 不要过度幼态 / 不要浓妆 / 不要大红唇 / 不要现代服装 / 不要皇冠 / 不要西方盔甲 / 不要霓虹 / 不要赛博朋克 / 不要复杂背景 / 不要多人物 / 不要文字水印]
```

### 已确认模板案例

**陆雪琪四视图角色参考表 v2** —— `archive_陆雪琪_四视图角色参考表_诛仙动画风_v2.png`

```text
项目与用途：第1集 / 角色参考表 / 角色：陆雪琪
画面规格：16:9 横版，同一张画布，4 子图横向并列
风格：腾讯《诛仙》动漫第二/三季 3D 国产动画风格、玄机科技建模路线、清冷写实系国漫美型

统一人物设定：陆雪琪，30 岁女性仙侠剑修，青云门小竹峰首座，张小凡之妻。
鹅蛋脸，柳叶弯眉，凤眼微挑，冰蓝色瞳孔，薄唇淡粉，肤白如雪。身形修长挺拔，7.5-8 头身，约 7.8 头身最优。
黑发及腰，部分用玉簪盘起，两侧长鬓发垂落。
服装：纯白长裙、纱质飘逸外袍、银丝暗纹、白丝绦、浅青银白少量边饰；清冷克制。

视图要求：
1. 正面 0 度，全身站立，双脚并拢，双臂自然下垂或微张 15-20 度
2. 左侧 90 度全身站立（必须真 90 度，不要 45 度斜侧面）
3. 背面 180 度全身站立（不要回头、不要侧身）
4. 人像特写：肩部以上，脸型与正面 100% 匹配

要求：前三个全身视图头顶脚底同一水平线，四个子图人物五官、肤色、发色、服装、发饰一致
8K 超清，电影级柔和光影，真实皮肤质感，禁止磨皮、塑料皮肤、网红脸
负面限制：不要换脸、尖下巴、过度幼态、浓妆、大红唇、性感姿势、暴露、透明服装、现代服装、皇冠、大量珠宝、西方盔甲、高跟鞋、大面积红色、霓虹、赛博朋克、复杂背景、多人物
```

完整原文见 `红鸾笑/角色资产/陆雪琪/角色设定.md` 的「00 · 四视图角色参考表」。

## 三、道具资产（资产库）

> 第1集尚未生成任何道具资产，下面是骨架模板，等天琊剑、碧瑶绿衣风铃、烧火棍等开始生图时填充。

### 用途

定调关键道具的剑光、布料、火焰、灯笼等视觉特征，作为多镜头共用的"道具母版"。

### 推荐提示词结构

```text
项目与用途：第N集 / 道具资产 / 道具：[道具名]
画面规格：16:9 横版，同一张画布，4 子图（正面 / 侧面 / 细节 / 持握或使用场景）
风格：[动画厂名] 建模路线 / 与全片风格一致

统一道具设定：[道具名]，[材质]，[尺寸感]，[关键视觉特征]。
[子图1 正面特征]，[子图2 侧面特征]，[子图3 细节特征]，[子图4 使用场景]。

要求：[四子图一致 / 8K / 柔和光影 / 真实材质质感]
负面限制：[不要 X / 不要 Y / 不要 Z]
```

### 待补案例（占位）

- **天琊剑**：碧蓝剑光、剑身修长、剑柄玉饰
- **碧瑶绿衣风铃**：浅水绿色薄绸残片、撕裂边缘、细绳系挂、手掌大小
- **烧火棍**：黑色烧火棍、第3集出场（先在第1集不出现）

## 四、资产命名与归档规范

| 资产 | 源文件位置 | HTML 引用副本 |
|------|-----------|---------------|
| 场景图 | `红鸾笑/场景资产/<地点>_<要点>.png/jpg` | `制作脚本/<集>/assets/<同名>.png/jpg` |
| 角色母版 | `红鸾笑/角色资产/<角色>/归档/archive_<角色>_<要点>_v<版本>.png` | `制作脚本/<集>/assets/character_<拼音>_<要点>_v<版本>.png` |
| 道具母版 | `红鸾笑/资产库/<道具>/归档/archive_<道具>_<要点>_v<版本>.png` | `制作脚本/<集>/assets/prop_<拼音>_<要点>_v<版本>.png` |

确认归档副本后，必须把路径同步到 `资产提示词库` 索引（见 `01_HTML制作脚本格式.md` 第 4.5 节）。
