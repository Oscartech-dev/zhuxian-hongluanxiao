## 1. 项目一句话

《红鸾笑》是基于《诛仙》后传气质延展的 AIGC 短剧项目。当前正在制作第1集「草庙村」，重点是陆雪琪从小竹峰出发，回到草庙村废墟，发现张小凡隐居痕迹，并在木屋前完成久别重逢。

## 7. 提示词写法规则

### 7.1 最重要流程规则

所有提示词先讨论，不要直接写入文件。

适用范围：

- VG 视频提示词
- 场景资产图提示词
- 角色资产提示词
- 道具资产提示词
- 视频素材生成提示词

流程：

1. 在对话中起草。
2. 根据用户反馈修改。
3. 用户明确确认后，才写入 `build_episode1_caomiaocun_storyboard.py`、模板 md、HTML 生成器或工作日志。
4. 写入后重新生成 HTML。
5. 检查 HTML 中提示词、资产和素材是否正确显示。

不要在用户未确认时改 md / html / py / 工作日志。

### 7.2 VG 视频提示词结构

参考：

`红鸾笑/分镜提示词/_模板/03_VG视频段提示词模板.md`

正式结构：

1. `【基本设定】`
2. `【氛围与面质】`
3. `分镜一 / 分镜二 / 分镜三...`

每个分镜建议包含：

- 镜头目的
- 景别
- 构图
- 机位与镜头
- 运镜手法
- 画面内容
- 声音

写法原则：

- 不写与生成无关的说明性语言，如“任务”“参考资产”“你需要”。
- 不把教程、方法论、备注混进视频提示词。
- 分镜描述要像给导演和摄影师看的镜头说明。
- 镜头语言要明确，例如 Slow Push-in、POV、Tilt-up、slow orbit、High Angle、低机位等。
- 情绪可以有小说笔法，但要转化为可见的微动作、眼神、呼吸、手部动作、衣袂、环境声。
- 静态段要慢，动作段才加速度感。
- 声音写同期声，不默认加 BGM，除非用户明确要求。

### 7.3 Seedance 增强写法

参考：

`红鸾笑/分镜提示词/_模板/03A_Seedance视频段增强写法.md`

当前可用技巧：

- 0-5 / 5-10 / 10-15 三段节奏法
- 每个分镜明确“镜头目的”
- 慢推、拉远、低角度仰拍、POV、跟拍、slow orbit、航拍俯瞰
- 速度段可以使用“前景快速掠过 / 中远景缓慢移动 / 速度由中速逐渐加快 / 降落时风声变急”
- 强风格技巧，如 Dutch Angle、Crash Zoom、强胶片颗粒、Dolly Zoom、Low Key Lighting 等，暂不适合第1集当前温柔克制气质，但已记录为未来工具箱。

### 7.4 资产图提示词规则

参考：

`红鸾笑/分镜提示词/_模板/02_资产图提示词模板.md`

原则：

- 一张资产图对应一个最终提示词。
- 资产提示词库按人物资产、场景资产、道具资产分类。
- 不要在文生图提示词中写无效说明，如“参考资产：某路径”“第1集草庙村某镜头使用”等。
- 场景母图尽量无人，尤其是用于后续分镜切分的主场景图。
- 场景图要写清机位、镜头、空间层次、核心物件、光线和负面限制。
- 人物图要强调角色一致性、视图排版、头身比、服装、脸型、发型和风格统一。
- 道具图要强调完整入画、材质、结构、比例、局部细节和后续视频可复用性。

### 7.5 用户偏好的写法

- 中文讨论时不要用代码块包长提示词，否则会出现横向滑块，阅读不方便。
- 对话中展示提示词时，用普通文本和小标题，让聊天框自动换行。
- 文字行间距要紧凑，类似第一版普通文本。
- 如果用户只是讨论，不要急着改文件。

### 8.1 VG-01 视频提示词范例：慢情绪段

VG-01 是当前最重要的基线范例，已跑通 Seedance。后续写情绪段、室内段、慢镜头段时优先模仿它。

关键结构：

- 先写 `【基本设定】`：人物、场景、声音。
- 再写 `【氛围与面质】`：风格核心、视觉基调、色彩与影调。
- 分镜按时间码写，每个分镜写景别、构图、机位、运镜、画面内容、声音。
- 情绪不直接喊“悲伤”，而写成“垂眼、唇线绷住、指节收紧、呼吸微滞、眼底犹豫沉下去”。
- 慢镜头不只写“慢”，而写“极慢横移，移动量小”“镜头非常慢地向内推，不打扰人物”“每一个动作都要慢，停顿要长”。

可模仿片段：

分镜三：00:07-00:11

景别：手部特写、面部近景连续组接。

构图：先近景特写拍她垂在木案旁的手，指尖轻触卷轴边缘或青瓷茶盏；再切到她的眼睫、唇线和侧脸进行特写，面部表情有呼吸感。

机位与镜头：手部特写段约 0.6 米俯拍机位，浅景深虚化背景；面部近景段约 1.2 米平视机位，85mm 定焦，浅景深虚化圆窗与纱帘。

运镜手法：慢镜头，85mm 近景，轻微推近；每一个动作都要慢，停顿要长，情绪靠细节出现。

画面内容：她的手指微微收紧，又松开，像在压住一个念头。晨光掠过她的眼睫，她垂眼片刻，唇线轻轻绷住。她低声喃喃，仿佛在对一个不在场的人说话：小凡……是你吗？你在哪里？说完后，她的呼吸微微一滞，眼底的犹豫慢慢沉下去，眼神反而变得清澈而坚定。人物面部严格参考陆雪琪四视图角色母版，白色仙门服饰、黑发、玉簪、清冷气质保持一致。

声音：低语，不加字幕。

### 8.2 VG-02 视频提示词范例：道具召唤段

VG-02 是“人物动作 + 道具响应 + 仙侠仪式感”的范例。后续写武器召唤、法宝飞来、角色从静到动时可以模仿。

关键写法：

- 不把复杂动作塞进一个镜头，而拆成：人物入画 -> 手部召唤特写 -> 道具独立响应 -> 道具入手/角色离开。
- 道具段要明确“画面中只出现天琊神剑，不出现其他人物”，减少模型混淆。
- 召唤动作要有力，但不战斗化。

可模仿片段：

分镜三：00:05-00:08

镜头目的：表现天琊神剑响应召唤，独立完成“震动、起飞、飞向陆雪琪”的视觉段落。

景别：天琊神剑特写到飞行动作镜头。

构图：画面保持只有天琊神剑，剑身与剑鞘放在室内木案或剑架上，周围有卷轴、青瓷茶盏、纱帘和晨光。

机位与镜头：近景特写，85mm 到 50mm；浅景深，剑身清晰，背景虚化。

运镜手法：先静止半秒，剑身轻微震动；随后剑身浮起，镜头短暂跟随剑身向外飞出，形成一道碧蓝流光。

画面内容：天琊神剑在木案或剑架上轻轻一震，剑鞘与剑身发出清冷微光。下一瞬，神剑离开原处，带着细细的碧蓝流光穿过室内晨光，掠过纱帘与圆窗，向回廊方向飞去。画面中只出现天琊神剑，不出现其他人物。

声音：剑鞘轻震声，清冷剑鸣，流光掠过空气的细微声响。

### 8.3 VG-03 视频提示词范例：速度感和降落段

VG-03 是“POV 飞行 + 俯冲 + 低机位降落 + 发现线索”的范例。后续写移动速度、飞行、降落、追踪时可以模仿。

关键写法：

- 速度感不是写“很快”，而写前景、中景、远景的相对运动。
- 空中画面不能只有空白云层，要有下方山脉、山谷、废墟轮廓作为空间参照。
- 降落后节奏要变慢，用慢环绕和回眸承接情绪。

可模仿片段：

分镜一：00:00-00:04

景别：第一人称 POV 飞行镜头。

构图：前方是流动云雾，画面边缘偶尔有碧蓝剑光掠过；远处和下方隐约可见青云山山脉、山谷、层叠云海，避免只有空白云层。

机位与镜头：第一人称 POV，35mm，模拟陆雪琪御剑前行；前景云雾快速掠过，中远景山峦缓慢移动。

运镜手法：POV forward flight，速度由中速逐渐加快；穿过厚云时有轻微云雾擦镜感。

画面内容：陆雪琪御剑穿入云海，云雾从两侧掠过，剑光在画面边缘形成一条细细碧蓝线。前方山峦若隐若现，下方青云山谷在云间闪过。飞行速度明显增强，但画面保持稳定，不要剧烈晃动。

声音：破风声增强，云雾呼啸声掠过，剑鸣很细。

### 8.4 场景资产提示词范例：小竹峰外景

场景母图的核心是建立空间，不是表现人物情绪。小竹峰外景已多轮确认，后续写仙门居所、山中雅舍、清晨环境时可以模仿。

关键写法：

- 明确“不要人物”，避免模型添加人物。
- 先写机位和用途：16:9、24mm、三分之二侧前方观察视角、适合作为后续分镜切分用主场景图。
- 用建筑语言压住模型偏差：小拱桥、圆窗、青瓦飞檐、回廊残灯、竹帘、纱帘。
- 写清反向：不要农舍、草堆、茅草屋顶、柴火堆、农具、直板桥、大红灯笼。

可模仿片段：

画面是小竹峰深处的仙门女子居所，不是农舍。机位采用平视略高机位，24mm 广角，三分之二侧前方观察视角；镜头位于竹林溪岸外侧，隔着溪流与小桥看向中景居所。画面需要宽广、干净、层次清晰，适合作为后续分镜切分用的主场景图。

前景是高挑修长的青竹、竹叶、溪边石阶和薄雾。溪流清浅，水面有晨雾缓慢漂浮。溪上必须是一座小巧精致的弧形拱桥，不要直板桥；桥可以是竹木与青石结合，桥身略拱，栏杆细致，有东方园林感和仙门雅致感，尺度小而灵秀。

中景是一座小竹峰清雅居所，建筑必须有修仙意境：青瓦或深灰瓦屋顶，屋脊轻微上扬，飞檐克制，竹木结构精致，墙面素雅，门窗有圆形月洞门、圆窗或圆形窗棂元素。房外回廊下挂着几盏小巧素雅的暖色灯笼，灯笼还微微亮着，像清晨天刚亮、多数人尚未醒来时残留的夜灯。

### 8.5 角色资产提示词范例：陆雪琪四视图

陆雪琪四视图是本项目人物一致性的母版。后续写人物定妆照或角色参考表时，优先模仿这个结构。

关键写法：

- 所有内容同一张画布，4 个子图横向并列。
- 三视图头顶和脚底对齐，正面/侧面/背面姿态一致。
- 第 4 个子图是上半身或人像特写，作为后续脸型一致性锚图。
- 人物介绍放在视图要求之前，避免每个视图重复。
- 限定 7.5-8 头身，约 7.8 头身最符合当前美感。
- 不加道具，道具另行生成。

可模仿摘要：

你是一位专业的角色设计师和概念艺术家。请基于参考图片生成一套标准角色参考表。所有内容一行排版，4 个子图横向并列，同一张画布，暗灰色背景。角色为 30 岁女性仙侠剑修，鹅蛋脸精致不尖削，柳叶弯眉，凤眼微挑，冰蓝色瞳孔，肤白如雪瓷肌质感，黑发及腰，部分玉簪盘起，两侧留长鬓发自然垂落，身形修长挺拔，7.5-8 头身，约 7.8 头身。服装参考原图白色仙侠动画服饰，不写汉服，不加道具。4 个视图为正面、左侧 90 度、背面、上半身特写；三视图头顶和脚底必须在同一水平线上。风格为腾讯《诛仙》动画第二季/第三季 3D 国产动画风格，玄机科技建模路线，电影级光影，梦幻光晕，无水印，无文字。

### 8.6 道具资产提示词范例：天琊神剑

天琊神剑是当前道具资产范例。后续写烧火棍、法宝、片名书法字等关键道具时，可以模仿“主体定义 + 材质 + 结构 + 排版 + 负向约束”的方式。

关键写法：

- 明确它是“道具设定图”，不是视频镜头。
- 道具要完整入画，便于后续参考。
- 对材质、比例、局部细节、剑鞘/剑身关系写清楚。
- 写“后续视频中需保持该图一致”。

可模仿片段：

主体是一柄清冷、修长、灵性极强的仙门神剑，属于陆雪琪的本命神兵。整体气质不是战斗化的凶器，而是清冷、圣洁、锋利、克制的仙剑。设计风格参考腾讯《诛仙》动画剧集第二季 / 第三季 3D 国漫质感，玄机科技建模路线，仙侠古风，高级 3D CG 道具设定图。

剑身修长轻薄，比例优雅，剑体以银白色、冰蓝色、浅青蓝色为主。剑刃像寒冰与玉石结合，半透明质感，边缘锋利但不血腥。剑身中央有一条细长的冰蓝色灵纹或浅蓝透明槽线，内部仿佛有很淡的灵光流动。

剑鞘必须出现。剑鞘与剑身风格统一，银白、冰蓝、浅青蓝为主，表面有细密银丝纹、云纹、寒冰纹或青云门仙纹。剑鞘修长轻薄，末端精致，带少量玉石嵌件。剑鞘和剑身可以并排展示，也可以剑身半出鞘展示，但必须清楚看到剑鞘结构，方便后续制作“剑身一震、带鞘飞起、响应召唤”的镜头。

  - 合并视频生成提示词
  - 合并视频素材区
  - 参考资产区
